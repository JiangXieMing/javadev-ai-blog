# About JavaDev AI

Hi, I'm a senior Java developer with 10+ years of experience building enterprise applications. I write about **practical AI integration** — the kind you can use in production today, not toy demos.

## What I Cover

- **LangChain4j & Spring AI** — Framework tutorials and comparisons
- **AI Agents** — Building intelligent tools with tool calling and RAG
- **Code Review Automation** — Using AI to improve code quality
- **Java Security** — SQL injection, XSS, and other vulnerabilities AI can detect
- **SaaS Building** — Lessons from building and scaling an AI tool as a solo developer

## Why This Blog

Most AI content falls into two categories: too theoretical (academic papers) or too basic ("Hello World with ChatGPT"). This blog fills the gap — practical, production-ready guides for Java developers who want to build real AI applications.

Everything I write about, I've built and deployed. No theoretical fluff.
